Manufacturing Pipeline

Privacy Policy

This policy was last modified on May 7th, 2026.

Manufacturing Pipeline is a volunteer-run project that is owned and published by the Littleton STEM Educational Foundation ("Littleton Robotics"), a non-profit organization based in the United States.

Information We Collect

Littleton Robotics does not collect data from users of the Manufacturing Pipeline application. Manufacturing Pipeline is a self-hosted application — all data entered by team members (part names, project details, notes, and manufacturing status) is stored exclusively in infrastructure provisioned and owned by each team. Littleton Robotics has no access to this data.

Third-Party Data Collection

The Manufacturing Pipeline repository is hosted by GitHub, which publishes a privacy policy that applies when interacting with these services.

The Manufacturing Pipeline OnShape panel integrates with the OnShape API to read part geometry and export STEP files. These interactions are covered by OnShape's privacy policy.

Sign-in to the manufacturing board uses Google Identity Services. These interactions are covered by Google's privacy policy.

Teams that deploy Manufacturing Pipeline use third-party infrastructure providers such as Vercel and Neon. Use of data within a team's deployment is subject to those providers' respective privacy policies.

Contact

Questions about this policy can be directed to software@team6328.org.

Manufacturing Pipeline is open-source software. Source code is available on GitHub.
